## Week 11: Project 2: Landing Page

- [GitHub repo](https://github.com/rozina-aamir/landing-page)
- [Live site](https://rozina-aamir.github.io/landing-page/)

Best viewed on 1351 × 556

This is a landing page created with HTML and CSS.

The aim of this was to learn more about structuring HTML and using CSS flexbox properties in attempt to make the site responsive and also look decent.

The website took me approximately 1 week to make, where I'd spend about 2-3 hours a day working on it.

I created this website as closely to the design specifications as possible.

Overall, this project felt a bit complex at first as I had to refer to the docs quite often, but once I gained an understanding of what each flexbox property did and when to use it, doing the rest of the project felt easier to do, even if I did get a bit carried away with tweaking the design to perfection.
