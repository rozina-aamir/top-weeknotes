## Week 20: Project 3: Rock, Paper, Scissors

- [GitHub repo](https://github.com/rozina-aamir/rps)
- [Live site](https://rozina-aamir.github.io/rps/)

This project is a simple game of Rock, Paper, Scissors. It's played between the user and the computer, where the results are logged to the console. The game itself lasts for 5 rounds and logs the final scores to the console.

When the computer makes a decision, it will randomly return "rock", "paper", or "scissors" as a string.

When the user makes a choice, they are expected to write "rock", "paper", or "scissors" into the prompt box that appears when the user clicks on the play button.

The scores are kept, and whoever scores the most points at the end of the game wins.

The aim of this was to learn more about the basics of Javascript, focusing on variables, operators, data types, conditionals, functions, problem solving, and understanding errors.

This took me approximately one month to make, where I'd spend about 30 minutes a few days a week working on it, including lots and lots of debugging.

Overall, this project felt extremely complex for me, and the docs were quite tricky to understand, since they covered a lot of things that I don't yet understand.
