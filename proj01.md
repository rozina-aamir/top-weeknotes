## Week 3: Project 1: Odin Recipes

- [GitHub repo](https://github.com/rozina-aamir/odin-recipes)
- [Live site](https://rozina-aamir.github.io/odin-recipes/index.html)

This is a simple and barebones recipe site created using just HTML.

The aim of this was to learn more about file structures, navigating the command line interface, creating files and directories using the command line, using git and GitHub, writing good commit messages, and very basic HTML things, such as knowing how to add images, link to other pages, and creating different types of lists.

The website took me approximately 1 week to make, where I'd spend about 30 minutes a day working on it.

The recipes were loosely based on the recipes found on Allrecipes.

Overall, I'd say doing this project was fairly straightforward since I previously knew about HTML, but it still felt like I learned a lot, mainly in terms of becoming more comfortable with using the command line.
